### **邮件系统**

&#8195;&#8195;该系统可以进行简单的收发邮件以及查看详情。

&#8195;&#8195;运行在Tomcat8.0之上。

- pom.xml中有该系统需要依赖的包，tomcat中的servlet-api包需要自己添加。
- src中的java为java代码，为本系统后台代码。
- src中的resources中文件为资源文件。
- src中的webapp下为前端相关代码。
